DeepEventMine for EventExtraction 使用说明
Juvenile @ Niu Juxin Lv2

 1k 字  4 分钟

本文参考该项目的Github官方文档，介绍了DeepEventMine工具的安装与使用。

DeepEventMine工具是NLP-EventExtraction领域的一个开源工具，用于针对生物学文本（或者科技文献）进行事件提取。

Github Link：https://github.com/aistairc/DeepEventMine

该项目基于论文DeepEventMine: End-to-end Neural Nested Event Extraction from Biomedical Texts, Bioinformatics, 2020.

Linux Environment

该工具需要在Linux环境下进行。推荐使用VMware安装Debian或者Ubuntu的虚拟机。本文的环境为VMware虚拟机中的Debian GNU/Linux 10 (buster)。

Debian官网网站：https://www.debian.org/

Ubuntu下载地址：https://ubuntu.com/download/desktop

Preparation

Python 3.6.5

该项目需要使用3.6.5版本的Python。

Python3.6.5版本下载地址：https://www.python.org/downloads/release/python-365/

Github程序

将该项目在Github上的Code下载下来，解压后进入解压的文件夹下。如果没有特殊说明，之后的终端操作全部在该文件夹下进行。

DeepEventMine模型下载

根据官方文档的介绍，该项目对下面的7个任务都进行了训练：

  1. cg: Cancer Genetics (CG), 2013
  2. ge11: GENIA Event Extraction (GENIA), 2011
  3. ge13: GENIA Event Extraction (GENIA), 2013
  4. id: Infectious Diseases (ID), 2011
  5. epi: Epigenetics and Post-translational Modifications (EPI), 2011
  6. pc: Pathway Curation (PC), 2013
  7. mlee: Multi-Level Event Extraction (MLEE)
